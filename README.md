# Trix

Want to include Basecamp's awesome [Trix WYSIWYG
editor](http://trix-editor.org) in your Ruby on Rails application?
You've come to the right place!

## Installation

Add this line to your application's Gemfile and bundle:

```ruby
gem 'trix'
```
